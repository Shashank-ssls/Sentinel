About this system — methodology
Model
Multinomial / binary logistic regression (scikit-learn). No neural nets.
Features
TF-IDF over word 1–2 grams + 10 structural features (length, digit count, uppercase ratio, link/currency/urgency counts, has-URL/phone/email). One shared pipeline for training and inference.
Model W
Trained on the Mendeley SMS corpus (largely Western-origin smishing).
Model A
Trained on Mendeley + a 70% split of a real Indian SMS set; the 30% India test is held out (never trained or TF-IDF-fit on).
Explanations
The model's own signed coefficients (logit = coefficient × feature value) — faithful, not a surrogate.
Scope
Binary unwanted vs legitimate. Not smishing-specific: "unwanted" spans scam and promotional spam.
Full model comparison — held-out macro-F1

One message, two detectors. A model trained only on Western SMS (W) versus the same model retrained with Indian SMS (A) — classifying unwanted vs legitimate.

Fully local; no network calls. Verdicts are unwanted vs legitimate, not smishing-specific. Aggregate metrics are read from the evaluation artifacts; per-message explanations are each model's own coefficients.
